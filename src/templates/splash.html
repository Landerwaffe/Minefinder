Disadvantages with the current system

Bias towards projects with money

Once a project has been uploaded, Minefinder will carry out verification, editing if needed, generate keywords and tags for further data analysis then publish the project to the platform.

Have no set format

Data in each project is set out differently, which makes it difficult to compare or even find the data you need to make a decision.

Payment doesn’t guarantee outcome

These websites require money with no guarantee of outcome or quality of projects and require serendipity of the perfect project listing on the system/website you use.

Profitable projects unseen

Valuable projects are discarded because of the higher barrier to entry. There’s profit to be had in taking less publicized projects.
